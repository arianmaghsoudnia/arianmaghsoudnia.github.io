Data-Dricen Energy Consumption Prediction Module

Machine Learning - Data Driven methods, Energy sytstems

The meteorological data of the LBNL-74 building is gathered. Using multiple sensors the data provided by the lab. Time-related features are guessed and added and then using data-driven methods of Cross-validation, Random Forrest, and online learning methods, the co-relation of gas and electricity consumption and the seasonal variation in consumption is predicted and verified by real data. The project is done using google collaboratory and Python 3 and is openly accessible on GitHub.
